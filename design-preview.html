Option A+ — カーボンスレート + シアン（推奨）

背景の紺の彩度を落としてスレートグレー寄りに（「水底感」を解消）。ボーダーを少し明るくしてパネルの境界を明確化。 シアンをヘッダー下線・現在セグメント・目標W値に使い、「今ここ」を一点集中で示す。アクセント色を意味のある場所にだけ使う設計。

Option A+ の変更ポイント（Option A との差分）

→ 背景の青みを減らしスレートグレー寄りに。「水の底」感が薄れる
→ ボーダーを #2c4055 に明るくしてパネルの境界を可視化。空間の構造が見えやすくなる
→ ヘッダー下に 2px シアンライン。アプリの「顔」になる一点
→ ワークアウト中「目標W」カードだけシアン枠＋シアン文字。他は白。「今何Wを目指すか」が即座にわかる
→ セグメントの現在行背景は acc-bg（シアンの極薄）。ゾーン色の左ボーダーはそのまま維持
→ プログレスバー（細い 4px）がシアンで fill。ライド中の進行感を主張せず示す
→ 接続済バッジをエメラルド（#34d399）に。緑のまま、より明るく清潔
→ アクセントは ヘッダー線・目標値・現在セグメント・プログレスバー の4箇所のみ。それ以外は白
